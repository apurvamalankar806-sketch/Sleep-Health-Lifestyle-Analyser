# sleep-lifestyle-eda-project
Exploratory Data Analysis on Sleep Health and Lifestyle dataset — uncovering how stress, BMI, occupation, and physical activity affect sleep quality.

This project performs an end-to-end Exploratory Data Analysis (EDA) 
on the Sleep Health and Lifestyle dataset (374 individuals).

We explore how factors like stress level, BMI, occupation, and physical 
activity influence sleep quality and duration — using Python, Pandas, 
Matplotlib, and Seaborn.

Key questions answered:
- Which occupation sleeps the best and worst?
- How strongly does stress affect sleep quality?
- Does physical activity improve sleep?
- What role does BMI play in sleep patterns?

Dataset source: Kaggle — Sleep Health and Lifestyle Dataset
